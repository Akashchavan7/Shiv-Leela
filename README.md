# Shiv Leela Invitation

A mobile-first invitation-style React web app built with Vite, React, TypeScript, Tailwind CSS, React Router, and Framer Motion.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Customize content

Update all invitation text, gallery media, links, and venue data in:

- `src/data/invitationData.ts`

## Deploy to GitHub Pages

This repo includes a GitHub Actions workflow that builds and deploys the app to GitHub Pages.

After pushing to the `main` branch:

1. Open the repository on GitHub.
2. Go to `Settings` -> `Pages`.
3. Set `Source` to `GitHub Actions`.
4. Push to `main` again if needed.

The Vite base path is already configured for the `Shiv-Leela` repository.
